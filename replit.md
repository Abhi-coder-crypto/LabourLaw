# Labour Codes

A multi-page business website for Labour Codes — an Indian labour law compliance consultancy.

## Stack
- **React 19** + **TypeScript** + **Vite 8**
- **Tailwind CSS v4** (via `@tailwindcss/vite`)
- **Framer Motion** for animations
- **React Router v7** for client-side routing
- **Lucide React** for icons

## Pages
- `/` — Home
- `/about` — About
- `/services` — Services listing
- `/services/:slug` — Service detail
- `/clientele` — Clientele
- `/resources` — Resources / blog
- `/careers` — Careers
- `/contact` — Contact

## Running the app
```bash
npm run dev
```
Runs on port 5000. The Vite dev server is configured for Replit (host `0.0.0.0`, `allowedHosts: true`).

## Other scripts
```bash
npm run build   # TypeScript compile + Vite production build
npm run lint    # Oxlint
npm run preview # Preview production build locally
```
